Recommendation Engines

What is a recommendation engine?

A recommendation engine is a machine learning system that attempts to predict user preferences based on the user's history, demographic information, the preferences of similar users, and content-based filtering. In recent years, recommendation engines have become an essential tool for any online business or community, allowing businesses to optimize user engagement and experience, therefore creating higher customer retention. Just a few examples of companies using recommendation engines are: Netflix, Facebook, Google, Amazon...

What are the different types of recommendation engines?

There two main types of recommendation engines, content-based filtering, and collaberative filtering. Both types have pros and cons as they vary in terms of information and computation needed to run algorithms. Most high-level implementations use a hybrid of these two systems.

Content-based Filtering

Content-based filtering is based solely on the comparison of one product to others. For example, if a Netflix user watches an action movie starring Keanu Reeves, the algorithm would examine feature vectors of that movie and return movies with similar feature vectors recommend to the user and will likely return additional action movies and/or additional movies starring Keanu Reeves. Algorithms used in content-based filtering use different methods to measure similarity in feature vectors that contain keywords, genres, etc.. Collaborative Filtering

Collaborative filtering uses user data to predict taste and future behavior. The algorithm can be as simple as comparing users' ratings of products or content, or as complex as using metadata and context awareness to target the use of advertising, content, or sales.
